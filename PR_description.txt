# feat: per-agent voice pilot — VoiceAdapter + OpenClaw demo (3 presets)

## Summary
This PR delivers the VoiceDNA per-agent voice pilot for OpenClaw. It adds an opt-in `VoiceAdapter` layer so agents can resolve to distinct VoiceDNA presets, plus a runnable demo and test coverage.

## Key points
- Opt-in only; default VoiceDNA behavior is unchanged.
- Per-agent routing resolves deterministically by agent id / agent name / default.
- Demo assets are checked in as WAVs for fast local validation.
- Existing public API behavior is preserved unless the new mapping is enabled.
